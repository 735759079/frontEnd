- float 脱离文档流和 position：absolute 不一样

  - 文字环绕的情况，是怎么形成的？float 之后，元素会按照脱离文档流来布局，文字不会。所以会形成文字环绕

- 分割线
  - 设置 height：0，再设置 border 值
  - 设置伪类：before 和 after 来设置，content 值，如下：
  ```
    .spliter::before,
    .spliter::after {
      content: "";
      display: inline-block;
      border-top: 1px solid black;
      width: 200px;
      margin: 5px;
    }
  ```
